# Earthquake Streamlit Dashboard

This project analyzes global earthquake data using a Streamlit dashboard.

## Project Description
The dashboard allows users to run SQL queries and explore earthquake data such as:
- Strongest earthquakes
- Deepest earthquakes
- Shallow earthquakes with high magnitude
- Most active earthquake years

The data is stored in a TiDB cloud database and accessed using SQL queries.

## Technologies Used
- Python
- Streamlit
- SQL
- Pandas
- SQLAlchemy
- TiDB Cloud
- USGS Earthquake API

## Files in this Repository
app.py – Streamlit dashboard application  
earthquake.ipynb – Data extraction and analysis notebook  
requirements.txt – Required Python libraries  
README.md – Project documentation

## How to Run the Project
Install dependencies:

pip install -r requirements.txt

Run the Streamlit app:

streamlit run app.py
